# NAME

unzip - Extract compressed files in a ZIP archive

# EXAMPLES

Extract zip file(s) (for multiple files, seperate file paths by spaces).

    unzip file(s)

Extract zip files(s) to given path.

    unzip files(s) -d /path/to/put/extracted/files
