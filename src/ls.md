# NAME

ls - list directory contents

# EXAMPLES

List all files, even hidden.

    ls -a

Sorting by size.

    ls -s

List all files with their rights, groups, owner.

    ls -ls

List all files with a prefix.

    ls prefix*

List all files with a suffix.

    ls *suffix

Sort files by last modified time, reversed.

    ls -tr
